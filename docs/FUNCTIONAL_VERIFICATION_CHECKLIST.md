## 1. 测试环境准备

### 1.1 环境启动

```bash
# 1. 安装依赖
pnpm install

# 2. 启动数据库
docker-compose up -d db redis

# 3. 执行数据库迁移
pnpm db:migrate

# 4. 填充测试数据
pnpm db:seed

# 5. 启动开发服务器
pnpm dev
```
