## 1. 系统简介

### 1.1 系统概述

设备护照追溯系统是一个面向机电自动化行业的B2B服务平台，主要功能包括：

- **设备全生命周期管理**: 通过数字护照追踪设备从采购到退役的全过程
- **专家服务撮合**: 智能匹配客户需求与专家技能，提供专业服务
- **供需对接**: 连接供应商与买家，实现高效的产品交易
- **信用评价体系**: 建立完善的评价和积分系统，保障服务质量

## 3. 账户管理

### 3.1 用户注册

#### 3.1.1 公司注册

1. 点击首页的"注册"按钮
2. 选择"公司注册"
3. 填写公司基本信息：
   - 公司名称
   - 组织代码（3位字母，用于护照码生成）
   - 公司类型
   - 法定代表人
   - 注册资本
   - 成立日期
   - 经营范围
4. 填写地址信息：
   - 注册地址
   - 实际经营地址
5. 上传营业执照
6. 填写管理员账户信息
7. 提交注册申请

#### 3.1.2 专家注册

1. 点击首页的"注册"按钮
2. 选择"专家注册"
3. 填写个人信息：
   - 姓名
   - 性别
   - 出生日期
   - 联系方式
   - 身份证号
4. 填写专业信息：
   - 专业领域
   - 技能标签
   - 工作经验年限
   - 服务区域
5. 上传证书和资质文件
6. 设置账户密码
7. 提交注册申请

### 3.2 用户登录

1. 访问系统首页
2. 点击"登录"按钮
3. 输入注册邮箱和密码
4. 点击"登录"按钮
5. 登录成功后自动跳转到对应的工作台

### 3.3 密码管理

#### 忘记密码
1. 在登录页点击"忘记密码"
2. 输入注册邮箱
3. 查收密码重置邮件
4. 点击邮件中的链接设置新密码

#### 修改密码
1. 登录系统后进入"个人设置"
2. 点击"修改密码"
3. 输入当前密码和新密码
4. 确认保存

## 4. 设备护照管理

### 4.1 创建设备护照

**操作角色**: 管理员、操作员

**操作步骤**:

1. 进入"设备护照"菜单
2. 点击"创建护照"按钮
3. 填写设备基本信息：
   - 设备名称
   - 设备型号
   - 产品类型（PLC/电机/传感器等）
   - 制造商信息
   - 序列号
4. 选择产地国家
5. 填写规格参数（可选）
6. 上传设备图片（可选）
7. 点击"创建"按钮
8. 系统自动生成护照码和二维码

**护照码格式示例**: `DP-MED-2601-PLC-DE-000001-A7`

### 4.2 查询设备护照

**操作步骤**:

1. 进入"设备护照"菜单
2. 使用搜索框输入关键词（护照码/设备名称/型号）
3. 使用筛选器按状态、类型过滤
4. 点击护照卡片查看详情

### 4.3 更新设备状态

**操作步骤**:

1. 进入护照详情页
2. 点击"更新状态"按钮
3. 选择新状态
4. 填写状态变更说明（可选）
5. 确认更新

**设备状态流转**:
```
创建 → 采购 → 质检中 → 质检通过 → 组装中 →
测试中 → 测试通过 → 已包装 → 运输中 →
已交付 → 服务中 → 维护中 → 退役
```

### 4.4 扫码查询

**公开用户操作**:

1. 使用手机摄像头或扫码器扫描设备二维码
2. 自动跳转到设备公开信息页面
3. 查看设备基本信息、状态、维护记录摘要

**登录用户操作**:

1. 扫码后可查看更详细的信息
2. 可直接发起服务请求
3. 可查看完整的服务历史

## 5. 专家服务平台

### 5.1 专家工作状态管理

专家有五种工作状态：

| 状态 | 图标颜色 | 说明 |
|------|---------|------|
| 抢单中 (RUSHING) | 红色/橙色 | 急需订单，优先接收推送 |
| 空闲 (IDLE) | 绿色 | 正常等待分配 |
| 已预定 (BOOKED) | 蓝色 | 已接单，等待服务开始 |
| 服务中 (IN_SERVICE) | 紫色 | 正在执行服务 |
| 休息中 (OFF_DUTY) | 灰色 | 暂不接单 |

**切换状态操作**:

1. 进入专家工作台
2. 在顶部状态栏选择目标状态
3. 确认切换

### 5.2 服务大厅

**功能说明**: 浏览所有公开的服务请求

**操作步骤**:

1. 进入"服务大厅"菜单
2. 浏览可接单的服务请求列表
3. 使用筛选器过滤：
   - 服务类型
   - 紧急程度
   - 预算范围
   - 服务地区
4. 点击感兴趣的服务请求查看详情

### 5.3 申请接单

**操作步骤**:

1. 在服务大厅或匹配结果中选择服务请求
2. 点击"申请接单"按钮
3. 填写申请信息：
   - 报价金额
   - 预计工时
   - 服务方案说明
   - 可开始时间
4. 提交申请
5. 等待客户确认

### 5.4 服务记录管理

**查看服务记录**:

1. 进入"服务记录"菜单
2. 查看所有服务记录列表
3. 使用状态筛选：待开始/进行中/已完成/已取消

**服务执行流程**:

│                        服务执行流程                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │

│     │ 点击"开始服务"                                                │

│  ② 进行中 (IN_PROGRESS)                                             │
│     │                                                               │

│     ▼                                                               │
│  ③ 已完成 (COMPLETED)                                               │
│     │                                                               │

### 5.5 查看服务记录详情

**操作步骤**:

1. 在服务记录列表点击"查看详情"
2. 查看完整服务信息：
   - 服务基本信息
   - 客户联系方式
   - 时间安排
   - 价格信息
   - 服务备注
   - 客户评价（如有）
3. 执行相应操作（开始/完成/取消）

**查看专家护照**:

## 6. 服务请求与工单

### 6.1 创建服务请求

**操作角色**: 客户

**操作步骤**:

1. 访问"服务请求"页面（可通过扫码直接访问）
2. 填写服务信息：
   - 服务标题
   - 问题描述
   - 服务类型（维修/保养/安装/检测等）
   - 紧急程度
3. 填写联系信息：
   - 联系人姓名
   - 联系电话
   - 服务地址
4. 设置预算范围（可选）
5. 上传图片附件（可选）
6. 关联设备护照（如有）
7. 提交请求

### 6.2 管理我的服务请求

**操作步骤**:

1. 进入"我的服务请求"菜单
2. 查看所有已提交的服务请求
3. 按状态筛选：草稿/已发布/进行中/已完成/已取消
4. 点击查看详情

### 6.3 查看专家申请

**操作步骤**:

1. 在服务请求详情页
2. 查看所有专家申请列表
3. 比较专家信息：
   - 专家评分
   - 报价金额
   - 服务方案
   - 历史评价
4. 点击"接受"选择专家
5. 其他申请自动拒绝

### 6.4 客户确认服务完成

**操作步骤**:

1. 进入"我的服务记录"菜单
2. 找到状态为"已完成"的记录
3. 查看服务报告和完成备注
4. 点击"确认完成"按钮
5. 服务正式结束，可提交评价

## 7. 评价系统

### 7.1 提交服务评价

**操作角色**: 客户

**前提条件**: 服务已确认完成

**操作步骤**:

1. 在服务记录详情页或"我的服务记录"
2. 点击"提交评价"按钮
3. 填写评分：
   - 整体评分（1-5星，必填）
   - 服务质量（1-5星，可选）
   - 沟通能力（1-5星，可选）
   - 准时程度（1-5星，可选）
   - 性价比（1-5星，可选）
4. 填写评价内容（文字描述）
5. 提交评价

### 7.2 查看评价

**专家查看收到的评价**:

1. 进入"我的评价"或服务记录详情
2. 查看所有客户评价
3. 查看评分统计

**客户查看专家评价**:

1. 在专家详情页
2. 查看历史评价列表
3. 查看平均评分

## 8. 积分系统

### 8.1 积分概述

积分系统用于激励用户活跃参与平台活动：

| 积分类型 | 说明 |
|---------|------|
| 奖励积分 | 完成任务获得，可用于兑换 |
| 信用分 | 影响用户等级和权益 |

### 8.2 查看我的积分

**操作步骤**:

1. 进入"我的积分"菜单
2. 查看积分概览：
   - 当前积分余额
   - 信用分
   - 信用等级
3. 查看积分交易记录
4. 了解积分获取规则

### 8.3 积分获取方式

| 行为 | 积分 | 说明 |
|------|------|------|
| 发布服务请求 | +10 | 每日上限3次 |
| 首次发布服务 | +50 | 新用户一次性 |
| 完成服务（专家） | +30 | 每次服务完成 |
| 获得5星好评 | +50 | 每次 |
| 邀请新用户注册 | +100 | 用户完成注册 |
| 被邀请人首单 | +200 | 被邀请人完成首单 |
| 设备接管成功 | +50 | 每次接管 |
| 连续7天登录 | +20 | 每周 |
| 完成实名认证 | +100 | 一次性 |

### 8.4 信用等级

| 等级 | 信用分范围 | 权益 |
|------|-----------|------|
| 青铜 | 0-199 | 基础功能 |
| 白银 | 200-499 | 优先客服 |
| 黄金 | 500-999 | 服务费9.5折 |
| 铂金 | 1000-1999 | 优先匹配，服务费9折 |
| 钻石 | 2000+ | 最高优先级，服务费8.5折 |

## 12. 常见问题

### 12.1 账户相关

**Q: 注册后多久能通过审核？**

A: 通常1-2个工作日内完成审核，审核结果会通过邮件通知。

**Q: 忘记密码怎么办？**

A: 在登录页点击"忘记密码"，通过注册邮箱重置密码。

**Q: 如何修改公司信息？**

A: 联系管理员或在公司设置页面提交修改申请。

### 12.2 服务相关

**Q: 如何取消已发布的服务请求？**

A: 在服务请求详情页点击"取消请求"，如已有专家接单需说明取消原因。

**Q: 专家不按时服务怎么办？**

A: 可以取消服务并提交投诉，平台会介入处理。
